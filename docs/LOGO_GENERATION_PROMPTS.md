## **Tool-Specific Tips**

### **Midjourney Best Practices**
- Start with `--style raw` for cleaner, less artistic results
- Use `--v 6` for the latest model (most accurate)
- Add `--no text, typography, letters` if it generates unwanted text
- Use `--chaos 0` for consistency, `--chaos 20` for variety
- Try `--stylize 100` (minimal) to `--stylize 750` (more artistic)
- Use `--repeat 4` to generate multiple variations efficiently
- Upscale your favorites with "U1", "U2", "U3", "U4" buttons
- Use "Vary (Subtle)" for refinements without major changes

### **DALL-E 3 Best Practices**
- Be extremely specific about dimensions and format
- Always request "transparent background" explicitly
- Mention "vector style" or "flat design" for cleaner edges
- Specify exact hex colors in the prompt
- Request "suitable for both light and dark backgrounds"
- If text is garbled, ask to "focus on icon/symbol only"
- Generate multiple variations, pick the best, then refine

### **LogoDiffusion / Looka / Brandmark**
- Use shorter, keyword-focused prompts
- Let the tool's AI suggest typography
- Generate 10-20 variations quickly
- Download SVG format when available
- Many tools offer instant color palette adjustments
- Some provide automatic brand kit generation

## **Iteration Strategy**

### **Phase 1: Initial Generation (2-4 hours)**
1. Generate 4-8 variations of **each style** (Neural, Circuit, Growth)
2. Generate both square icon and horizontal lockup versions
3. Test different aspect ratios and compositions
4. Save all promising candidates

### **Phase 2: Selection & Refinement (1-2 hours)**
1. Select top 3-5 designs from all variations
2. Test at different sizes (512px down to 32px)
3. Check readability on light and dark backgrounds
4. Get feedback from stakeholders
5. Narrow down to 1-2 final candidates

### **Phase 3: Final Polish (1-2 hours)**
1. Regenerate chosen design(s) with specific refinements
2. Request all size variations (see Export Specifications)
3. Generate dark mode and monochrome versions
4. Create social media assets
5. Ensure perfect scalability

### **Phase 4: File Preparation (1 hour)**
1. Optimize PNG files (use TinyPNG.com or similar)
2. Minify SVG files (use SVGOMG.net)
3. Organize files by size and format
4. Test all formats in browser/app
5. Prepare final delivery package

## **Quality Checklist**

Before finalizing, verify:

- [ ] Logo is recognizable at 16x16px (favicon size)
- [ ] Logo scales perfectly from 16px to 512px+
- [ ] Colors match exactly: `#3b82f6` and `#10b981`
- [ ] Design works on white, black, and colored backgrounds
- [ ] No pixelation or artifacts in any size
- [ ] SVG files are clean and optimized (<10KB)
- [ ] PNG files have true transparency (not white background)
- [ ] Dark mode version has sufficient contrast
- [ ] Monochrome versions are legible
- [ ] Horizontal lockup maintains balance and spacing
- [ ] Icon-only version works independently
- [ ] Design feels modern, professional, and timeless
- [ ] Avoids trendy effects that will date quickly

## **Questions or Issues?**

If generated logos need refinement:
1. Save the original prompt used
2. Note specific issues (color, proportion, style)
3. Adjust prompt with explicit corrections
4. Regenerate with refined parameters

**Common fixes:**
- "Too complex" → Add "minimal, simple, clean" to prompt
- "Wrong colors" → Specify exact hex codes
- "Unwanted text" → Add `--no text, letters, typography`
- "Too artistic" → Add "flat design, vector style, geometric"
- "Not scalable" → Emphasize "simple shapes, bold lines, clear form"
